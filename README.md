# ARMOR (IRTF) — Slide Decks

This repository hosts public slide decks and talk materials for **ARMOR — Adaptive Resilience and Measurement for Operational Robustness**. These slides support classes, workshops, and mock IETF/IRTF sessions.

**Discussion:** [armor@irtf.org mailing list](https://mailman3.irtf.org/mailman3/lists/armor@irtf.org)  
**Policies:** IETF/IRTF [Note Well](https://www.ietf.org/about/note-well/)

## What ARMOR focuses on
- Practical resilience against network interference and tampering
- Measurements that inform engineering, and engineering that improves reachability
- Cross-protocol considerations (DNS, TLS/ECH, HTTP/MASQUE, transport)
